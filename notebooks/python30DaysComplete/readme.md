## 학습 방식

파이썬을 처음 배우는 사람이 작은 코드 실행, 관찰, 직접 재작성, 말로 설명하기를 반복하며 기초 문법을 실제 사용 능력으로 바꾸는 30일 코스입니다.

- 오늘의 목표와 이유를 먼저 읽고, 새 문법이 어떤 문제를 해결하는지 확인합니다.
- 예제 코드는 실행하기 전에 결과를 먼저 예상합니다.
- 실행 후에는 어떤 줄이 어떤 결과를 만들었는지 한 문장으로 설명합니다.
- 연습 셀은 예제를 보지 않고 직접 다시 작성합니다.
- 완료 기준을 스스로 확인한 뒤 다음 Day로 넘어갑니다.
