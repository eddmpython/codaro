## 학습 방식

1. 설명을 읽고 바로 아래 예제 코드를 실행합니다.
2. 출력이 왜 그렇게 나오는지 말로 설명합니다.
3. 연습 셀은 비워져 있으므로 직접 다시 작성합니다.
4. 다음 Day로 넘어가기 전에 같은 예제를 다른 값으로 한 번 바꿔봅니다.
