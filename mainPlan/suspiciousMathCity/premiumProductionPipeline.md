# 프리미엄 제작 파이프라인

## 1. 목적

허접하게 만들지 않기 위한 제작 기준이다. 품질은 이미지 양이 아니라 상태가 읽히는 도시, 직접 만지는 수학, 접근 가능한 조작감, 반복 가능한 검수에서 나온다.

## 2. 제작 단계

| 단계 | 산출물 | 통과 기준 |
| --- | --- | --- |
| `graybox` | CSS/SVG 지도, 시계탑 장면, 하단 조작판 | 360px, 키보드, 큰 글자, 모션 축소 통과 |
| `state-art` | 장소 상태 4종, 오브젝트 상태, 능력 아이콘, 단서 카드 | asset manifest 등록 |
| `character-pass` | 노바 표정, 제로 쪽지, 완료 리캡 컷 | 퍼즐보다 앞서지 않음 |
| `polish-pass` | 색, 질감, 미세 모션, 복구 전후 비교 | 시각 품질 보강, 레이아웃 유지 |
| `export-pass` | 원본, 최적화 SVG, 1x/2x bitmap, 접근성 이름 | runtime은 asset id만 사용 |

`graybox`가 통과하지 못하면 아트 polish로 넘어가지 않는다.

## 3. 프리미엄 화면 기준

| 영역 | 기준 |
| --- | --- |
| 지도 | 360px에서 장소 3-6개가 구분됨 |
| 장소 상태 | `locked`, `available`, `corrupted`, `restored`, `revisitable`이 색+형태+텍스트 중 2개 이상으로 구분 |
| 장면 | 핵심 단서 2개가 같은 화면에 보임 |
| 조작 | 버튼, 타일, 오브젝트가 손가락과 키보드로 모두 조작 가능 |
| 캐릭터 | 관찰과 감정만 돕고 퍼즐을 가리지 않음 |
| 하단 시트 | `peek`, `half`, `full` 상태 |
| 모션 | focus 300ms, wrong 180ms, correct 240ms, 완료 1-2초 |

## 4. Visual QA

뷰포트:

```text
360x740
390x844
414x896
768x1024
1280x800
```

캡처 상태:

| 상태 | 확인 |
| --- | --- |
| fresh map | 첫 화면이 빈 화면이 아님 |
| episode start | 장면 단서와 첫 질문이 보임 |
| hint open | 힌트가 장면을 가리지 않음 |
| wrong feedback | 오개념 반응이 보임 |
| restored map | 복구 전후가 구분됨 |
| ability drawer | 능력 이름, 개념, 쓰는 곳이 보임 |
| clue journal | 단서가 의미와 연결 장소를 가짐 |
| settings | 큰 글자, 모션 줄이기, 초기화 접근 가능 |
| largeText | 가로 overflow 없음 |
| reducedMotion | 완료 연출이 정지 상태로 대체됨 |
| storage blocked | 저장 실패 문구와 계속 진행 가능 |

불합격:

- blank root.
- console error.
- horizontal overflow.
- 버튼 텍스트 잘림.
- 하단 시트가 핵심 장면을 완전히 가림.
- SVG 상태 수가 예상과 다름.
- 색만으로 상태를 구분.

## 5. 금지선

- 에피소드 수를 품질보다 앞세우지 않는다.
- 문제 카드만 쌓지 않는다.
- 지도 전체를 통 bitmap으로 고정하지 않는다.
- 핵심 조작 오브젝트를 이미지로만 만들지 않는다.
- 별, 점수, 등급, 연속 기록, 순위, 광고형 보상으로 몰입을 만들지 않는다.
- 색, 소리, 빠른 움직임만으로 정보를 전달하지 않는다.
- 첫 시계탑 장면이 약한 상태에서 시즌 확장으로 넘어가지 않는다.

## 6. 아트 원천 관리

| 항목 | 규칙 |
| --- | --- |
| 원본 | 수정 가능한 원천 파일을 별도 보관 |
| runtime | 최적화된 SVG/bitmap만 사용 |
| 접근성 | asset마다 이름, 상태, 대체 정보 기록 |
| 권리 | 외부 자산 금지, 생성/제작 출처 기록 |
| print | 도서 확장 가능 자산은 2x 이상 또는 vector 우선 |

## 7. 첫 제품 컷

첫 제품 컷은 `시계탑이 멈춘 날`이다.

품질 목표:

```text
지도에서 멈춘 시계탑이 보인다.
시계와 버스 시간표가 같은 장면에 보인다.
아이의 조작으로 20분 간격이 드러난다.
시계탑과 정류장이 복구된다.
제로의 0 단서가 다음 이야기로 이어진다.
```

이 장면이 제품의 약속을 증명하지 못하면 다른 에피소드로 넘어가지 않는다.
