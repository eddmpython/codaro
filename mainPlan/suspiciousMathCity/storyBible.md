# 스토리 바이블

> 리부트 메모: 세계관과 캐릭터는 유지한다. 단, 첫 플레이 동기는 제로의 추상 미스터리가 아니라 고장 난 장치 수리다. 제로 단서는 시계버스 수리 성공 뒤 다음 장치를 예고하는 역할로 낮춘다.

## 1. 세계관

어느 날 도시의 `개념 코어`가 망가지면서 수학 개념이 도시에서 빠져나간다. 숫자는 사라지고, 길이는 맞지 않고, 빵은 공평하게 나뉘지 않고, 시계는 멈춘다.

도시는 완전히 무너진 곳이 아니다. 아이가 개념을 하나씩 되찾을 때마다 밝아지고, 연결되고, 움직인다.

## 2. 중심 미스터리

처음에는 `제로`가 도시를 망가뜨린 것처럼 보인다. 하지만 시즌이 진행될수록 제로는 수학을 싫어하는 악당이 아니라, 개념을 잃어버려 세계를 읽지 못하게 된 아이였다는 단서가 드러난다.

큰 비밀:

```text
도시를 고치는 일은 제로를 이기는 일이 아니라,
제로가 잃어버린 개념을 함께 되찾는 일이다.
```

초등학생 대상이므로 무섭거나 어둡게 가지 않는다. 이상하고 궁금하지만 안전한 모험이어야 한다.

## 2.1 수학 미스터리 문법

단서는 정답 보상이 아니라 개념으로 다시 읽히는 문장이다. 처음에는 수수께끼처럼 보이고, 에피소드의 개념을 사용한 뒤에는 도시의 규칙을 설명하는 문장으로 바뀐다.

단서 카드 포맷:

```md
- 겉보기 문장:
- 아이가 처음 느끼는 궁금증:
- 필요한 개념:
- 개념을 배운 뒤의 뜻:
- 다음 에피소드에서 만드는 이상현상:
```

제로는 수학을 싫어하는 아이가 아니라, 수학 개념이 빠진 세계를 잘못 읽는 아이다. 제로의 장난은 항상 하나의 오개념에서 출발한다.

Core Slice 오개념:

| 에피소드 | 제로의 오개념 | 도시 이상현상 |
| --- | --- | --- |
| 시계탑 | 0을 "아무것도 없음"으로만 알고, 시간 사이의 빈칸을 보지 못함 | 2시 40분과 3시 사이의 남은 시간이 사라짐 |
| 숫자 버스 | 10에서 0을 지워도 큰 차이가 없다고 생각함 | 10번 버스가 1번처럼 보이고 노선이 꼬임 |
| 반쪽 빵집 | 공평함을 조각 개수로만 보고 크기를 보지 못함 | 케이크 조각 수는 맞지만 크기가 다름 |

## 3. 주요 캐릭터

캐릭터의 상세 캐논은 [ipCanon.md](ipCanon.md)를 따른다. 앱과 도서에서 캐릭터 역할과 말투가 달라지면 안 된다.

### 아이 탐험가

사용자 자신이다. 이름 입력은 받지 않는다. 대화에서는 "탐험가"라고 부른다.

역할:

- 도시의 이상현상을 직접 관찰한다.
- 노바의 질문에 답한다.
- 개념 능력을 사용해 세계를 고친다.

### 노바

도시 안내 로봇이다. 정답을 바로 말하지 않고 관찰을 돕는다.

말투:

- 짧다.
- 친절하다.
- 힌트는 장면을 보게 만든다.

예:

```text
이상해. 버스 시간표는 3시를 가리키는데, 시계탑은 2시 40분에서 멈췄어.
두 시각 사이를 보면 출발 시간을 찾을 수 있을 것 같아.
```

절대 하지 않는 행동:

- 정답을 바로 말하지 않는다.
- 아이를 혼내지 않는다.
- 긴 강의를 하지 않는다.
- 점수나 순위로 동기를 만들지 않는다.

### 제로

미스터리 캐릭터다. 처음에는 장난처럼 보이는 흔적을 남긴다.

말투:

- 짧은 수수께끼.
- 개념이 빠진 문장.
- 아이가 다음 사건을 궁금해하게 하는 단서.

예:

```text
0이 없으면 아무것도 없어진 걸까?
아니면 무언가를 셀 수 없게 된 걸까?
```

절대 하지 않는 행동:

- 무서운 악당처럼 위협하지 않는다.
- 아이를 조롱하지 않는다.
- 단순 방해꾼으로 끝나지 않는다.
- 시즌 질문과 무관한 장난을 하지 않는다.

지면 등장 규칙:

- 앱에서는 쪽지, 번호판 흔적, 복구 전후의 작은 실루엣으로 등장한다.
- 스토리북에서는 마지막 1-2쪽에 감정 단서로 등장한다.
- 워크북에서는 정답 캐릭터가 아니라 오개념을 되돌아보는 질문으로 등장한다.

## 4. 도시 구역

| 구역 | 중심 개념 | 이상현상 |
| --- | --- | --- |
| 시계탑 광장 | 시각, 덧셈 | 시계가 멈추고 버스가 늦어진다 |
| 숫자 버스 정류장 | 수 배열, 규칙 | 버스 번호가 사라지고 노선이 꼬인다 |
| 반쪽 빵집 | 나눗셈, 분수 | 빵이 불공평하게 나뉜다 |
| 길이 다리 | 길이, 덧셈 | 다리 조각의 길이가 맞지 않는다 |
| 패턴숲 | 규칙, 반복 | 길이 반복되지만 표지판이 빠져 있다 |
| 도형 공방 | 도형, 각도 | 문과 톱니가 돌아가지 않는다 |
| 데이터 항구 | 표, 그래프 | 물건 수 기록이 사라진다 |

Core Slice는 앞의 세 구역만 연다.

## 5. 시즌 구조

### 시즌 1: 사라진 숫자

중심 개념:

- 수 세기
- 덧셈
- 뺄셈
- 시각
- 규칙
- 도형 기초

시즌 질문:

```text
숫자가 사라지면 도시는 무엇을 잃어버릴까?
```

### 시즌 2: 반쪽 도시

중심 개념:

- 곱셈
- 나눗셈
- 분수
- 길이
- 넓이

시즌 질문:

```text
전체와 부분을 모르면 공평함도 사라질까?
```

### 시즌 3: 데이터 항구의 음모

중심 개념:

- 표
- 그래프
- 평균
- 가능성
- 비율 기초

시즌 질문:

```text
기록을 읽지 못하면 진실도 사라질까?
```

## 6. Product Core Slice 에피소드 상세

### Episode 1: 시계탑이 멈춘 날

도입:

```text
도시 중앙의 시계탑이 2시 40분에서 멈췄다.
버스는 3시에 출발해야 하지만, 아무도 얼마나 기다려야 하는지 모른다.
```

갈등:

- 시계탑 바늘이 움직이지 않는다.
- 버스 기사들이 출발 시간을 놓친다.
- 광장 사람들의 약속이 뒤섞인다.

개념 순간:

```text
노바: 지금 시각과 출발 시각을 함께 보면, 남은 시간을 알 수 있어.
```

성공 변화:

- 시계탑 바늘이 다시 움직인다.
- 버스 정류장 불이 켜진다.
- 시간 렌즈가 도감에 추가된다.

단서:

```text
시계탑 아래에서 작은 쪽지가 발견됐다.
"0은 사라지는 숫자가 아니야. 자리를 지키는 숫자야."
```

예고:

```text
밤이 되자 숫자 버스의 번호판에서 0이 사라졌다.
```

### Episode 2: 숫자 버스의 실종

도입:

```text
버스 번호판이 흐릿해졌다.
2번, 4번, 6번 버스는 보이지만 다음 버스 번호가 비어 있다.
```

갈등:

- 버스가 잘못된 정류장으로 간다.
- 정류장 표지판 순서가 꼬인다.
- 사람들은 어느 버스를 타야 할지 모른다.

개념 순간:

```text
노바: 번호가 아무렇게나 있는 게 아니야. 같은 만큼 커지는 규칙을 찾아봐.
```

성공 변화:

- 버스 번호가 복구된다.
- 노선도가 이어진다.
- 패턴 렌즈를 얻는다.

단서:

```text
버스 뒷면에는 이런 문장이 적혀 있다.
"규칙을 모르면 길도 기억을 잃어."
```

예고:

```text
복구된 노선 끝에서 찢어진 도시 지도가 발견됐다.
```

### Episode 3: 반쪽 빵집

도입:

```text
빵집 주인은 케이크를 나누려 하지만, 조각 크기가 계속 달라진다.
손님 4명은 모두 같은 양을 받고 싶어 한다.
```

갈등:

- 어떤 손님은 큰 조각을 받고 어떤 손님은 작은 조각을 받는다.
- 진열대의 빵들이 반쪽만 보인다.
- 빵집 간판의 원 모양이 찌그러져 있다.

개념 순간:

```text
노바: 공평하게 나누려면 전체를 같은 크기의 부분으로 봐야 해.
```

성공 변화:

- 케이크 조각이 같은 크기로 나뉜다.
- 빵집 간판이 복구된다.
- 나누기 칼을 얻는다.

단서:

```text
계산대 아래에서 제로의 편지가 발견됐다.
"나는 왜 항상 남은 조각만 받는 걸까?"
```

예고:

```text
편지 뒤에는 다리 그림이 있었다. 다리는 딱 절반만 그려져 있었다.
```

## 7. 이야기 톤

좋은 톤:

- 따뜻한 미스터리
- 작은 도시 모험
- 직접 고치는 성취감
- 수학을 싫어하는 아이도 들어올 수 있는 문장

피할 톤:

- 시험지 말투
- 장황한 설명
- 과한 판타지 고유명사
- 무서운 악당
- 성인용 추리물 분위기

## 8. 문장 규칙

- 한 대사는 2문장 이하.
- 화면 한 장면의 설명은 80자 안팎.
- 새 개념을 설명할 때는 먼저 장면을 말하고, 그 다음 용어를 말한다.
- "공부", "시험", "점수" 같은 단어는 기본 흐름에서 쓰지 않는다.
- 수학 용어는 숨기지 않되, 등장 직후 행동으로 연결한다.

## 9. 앱과 도서 공통 규칙

같은 에피소드가 앱과 책에서 다르게 보일 수는 있지만, 줄거리는 갈라지지 않는다.

| 항목 | 앱 | 도서/워크북 |
| --- | --- | --- |
| 이상현상 | 조작 가능한 장면 | 큰 컷과 찾아보기 질문 |
| 노바 | 힌트와 focus 유도 | 발문 말풍선 |
| 제로 | 단서와 흔적 | 쪽지와 감정 단서 |
| 능력 | 버튼과 화면 효과 | 손동작, 타일, 색칠 |
| 복구 | 지도 상태 변화 | 전후 비교 컷 |

도서에서 먼저 확정한 설정이 앱의 registry와 맞지 않으면 캐논 위반이다.
