# 세계관 캐논

## 1. 목적

이 문서는 `수상한 수학도시`의 세계관, 캐릭터, 말투, 시즌 질문을 관리한다. 앱, 도서, 워크북, 활동지가 서로 다른 작품처럼 갈라지는 것을 막는다.

## 2. 핵심 문장

```text
수학 개념을 배우면 세계의 숨은 규칙이 보이고,
그 규칙으로 도시를 고치며,
제로가 잃어버린 개념을 함께 되찾는다.
```

## 3. 변경하지 않는 세계관 규칙

| 규칙 | 설명 |
| --- | --- |
| 도시는 무너진 세계가 아니다 | 아이가 고치면 밝아지고 연결되는 세계다 |
| 이상현상은 오개념에서 온다 | 단순 장난이나 악당의 공격이 아니다 |
| 제로는 이겨야 할 적이 아니다 | 개념을 잃어버린 아이에 가깝다 |
| 노바는 정답을 말하지 않는다 | 관찰할 곳을 알려준다 |
| 능력은 보상이 아니다 | 세계를 읽고 조작하는 문법이다 |
| 단서는 수집물이 아니다 | 배운 개념으로 다시 읽히는 문장이다 |

## 4. 캐릭터 캐논

### 탐험가

| 항목 | 규칙 |
| --- | --- |
| 정체성 | 사용자 자신 |
| 이름 입력 | 받지 않음 |
| 호칭 | 탐험가 |
| 역할 | 관찰, 선택, 조작, 복구 |
| 금지 | 성적, 점수, 순위로 평가받는 존재로 만들지 않음 |

### 노바

| 항목 | 규칙 |
| --- | --- |
| 역할 | 안내 로봇, 관찰 유도자 |
| 말투 | 짧고 다정하며 구체적 |
| 할 일 | 볼 곳을 알려주고, 오개념을 안전하게 되돌림 |
| 하지 않는 일 | 정답을 바로 말함, 혼냄, 긴 설명 |
| 지면 등장 | 말풍선 1-2문장, 발문 중심 |

예:

```text
시계와 시간표를 같이 봐.
두 시각 사이에 빈 칸이 있어.
```

### 제로

| 항목 | 규칙 |
| --- | --- |
| 역할 | 미스터리 캐릭터, 오개념의 얼굴 |
| 말투 | 짧은 수수께끼 |
| 감정 | 처음엔 장난처럼 보이지만 점점 외로움과 오해가 드러남 |
| 하지 않는 일 | 무섭게 위협함, 벌을 줌, 아이를 조롱함 |
| 지면 등장 | 쪽지, 흔적, 마지막 장면의 작은 실루엣 |

예:

```text
0이 없으면 아무것도 없어진 걸까?
아니면 자리가 사라진 걸까?
```

## 5. 시즌 캐논

| 시즌 | 제목 | 시즌 질문 | 중심 감정 |
| --- | --- | --- | --- |
| 1 | 사라진 숫자 | 숫자가 사라지면 도시는 무엇을 잃어버릴까? | 이상함과 발견 |
| 2 | 반쪽 도시 | 전체와 부분을 모르면 공평함도 사라질까? | 공평함과 회복 |
| 3 | 데이터 항구의 음모 | 기록을 읽지 못하면 진실도 사라질까? | 추리와 판단 |

시즌 finale는 계산량으로 해결하지 않는다. 단서 연결과 개념 이해로 해결한다.

## 6. 에피소드 캐논

에피소드 하나는 다음을 가진다.

| 항목 | 규칙 |
| --- | --- |
| 큰 사건 | 장소 하나의 이상현상 |
| 새 개념 | 1개 |
| 단서 | 1개 |
| 닫힌 답 | 이번 장소가 왜 이상했는지 |
| 열린 질문 | 다음 장소로 이어지는 질문 |
| 제로 오개념 | 하나의 잘못된 생각 |
| 능력 재사용 | 후속 에피소드 또는 재방문에서 1회 이상 |

## 7. 금지 변형

- 제로를 단순 악당으로 고정하지 않는다.
- 노바를 강의하는 선생님처럼 만들지 않는다.
- 탐험가를 시험 치는 학생처럼 만들지 않는다.
- 수학 능력을 레벨, 희귀도, 강화 아이템으로 바꾸지 않는다.
- 시즌마다 세계 규칙을 새로 갈아엎지 않는다.
- 도서와 앱의 줄거리를 따로 만들지 않는다.
