# 라이브 수리 스테이지 명세

## 1. 정의

라이브 수리 스테이지는 수상한 수학도시의 핵심 플레이 단위다. 문제를 읽고 정답을 제출하는 화면이 아니라, 고장 난 장치에 수학 부품을 끼우고 작동시켜 보는 장면이다.

```text
장치
부품
슬롯
능력
작동
반응
수리 기록
```

이 7개가 없으면 수리 스테이지가 아니다.

## 2. 스테이지 계약

```ts
type RepairStage = {
  stageId: string;
  title: string;
  brokenDevice: string;
  conceptId: string;
  abilityId: string;
  visibleClues: string[];
  parts: RepairPart[];
  slots: RepairSlot[];
  runButtonLabel: "작동";
  successRule: RepairRule;
  wrongReactions: WrongReaction[];
  representationLadder: RepresentationStep[];
  transferStageId?: string;
};
```

필수 필드 의미:

| 필드 | 의미 |
| --- | --- |
| `brokenDevice` | 아이가 고쳐야 할 장치 |
| `visibleClues` | 문제 전 장면에서 보이는 단서 |
| `parts` | 손으로 집을 수 있는 수학 부품 |
| `slots` | 부품을 끼울 장치 위치 |
| `successRule` | 장치가 작동하는 규칙 |
| `wrongReactions` | 잘못 끼웠을 때 보이는 이유 |
| `representationLadder` | 장면에서 수학 문장까지 이어지는 사다리 |
| `transferStageId` | 같은 능력을 한 번 더 쓰는 가까운 장치 |

## 3. 상태 흐름

```text
idle
→ partSelected
→ partPlaced
→ readyToRun
→ running
→ success | recoverableWrong
→ repaired
```

`recoverableWrong`은 실패 화면이 아니다. 장치가 왜 안 맞는지 보여주고 다시 배치 상태로 돌아간다.

## 4. 화면 구성

### 장치 스테이지

- 화면에서 가장 큰 영역.
- 핵심 오브젝트는 버튼 또는 포커스 가능한 요소로 구현.
- 슬롯은 비어 있는 이유가 보이게 표시.
- 능력을 켜면 숨은 구조가 드러남.

### 부품 트레이

- 부품은 숫자 카드가 아니라 장치 부품처럼 보인다.
- 부품은 손가락으로 누르기 쉬워야 한다.
- 사용한 부품은 슬롯에 고정되지만 다시 뺄 수 있다.

### 작동 버튼

- 이름은 `확인`이 아니라 `작동`.
- 누르면 장치가 움직이며 판정된다.
- 키보드에서는 `Enter` 또는 `Space`로 작동 가능.

## 5. 피드백 규칙

| 상황 | 장치 반응 | 문구 |
| --- | --- | --- |
| 부족 | 목표 직전에서 멈춤 | 아직 빈칸이 남았어 |
| 과다 | 목표를 지나쳤다가 되돌아옴 | 너무 많이 넣었어 |
| 다른 종류 | 연결선이 어긋남 | 이 부품은 여기 규칙과 맞지 않아 |
| 순서 오류 | 부품이 서로 자리 바꿈 | 앞뒤 간격을 다시 봐 |
| 성공 | 장치가 자연스럽게 작동 | 규칙이 맞아서 장치가 움직였어 |

문구는 45자 안팎으로 유지한다.

## 6. 첫 수리 스테이지: 시계버스

| 항목 | 내용 |
| --- | --- |
| stageId | `clockBus-01` |
| 장치 | 멈춘 시계버스 장치 |
| 개념 | 시각과 시간 차이 |
| 능력 | 시간 렌즈 |
| 단서 | 현재 2:40, 출발 3:00 |
| 부품 | 10분 부품 3개 |
| 슬롯 | 2:40->2:50, 2:50->3:00 |
| 작동 성공 | 슬롯 2개에 10분 부품 2개 |
| 부족 반응 | 버스가 2:50에서 멈춤 |
| 과다 반응 | 버스가 3:10까지 갔다가 돌아옴 |
| 수학 문장 | 10분 + 10분 = 20분 |
| 전이 | 5:20->5:40 정류장 전광판 |

## 7. 후속 수리 스테이지

| 순서 | 장치 | 개념 | 부품 |
| --- | --- | --- | --- |
| 2 | 0이 빠진 번호판 | 0과 자리값 | 0 부품, 자리 슬롯 |
| 3 | 번호 버스 | 규칙 | 2씩 증가 번호 부품 |
| 4 | 공평 식탁 | 공평 배분 | 빵 조각과 접시 |
| 5 | 케이크 분할판 | 단위분수 | 같은 크기 절단선 |

## 8. 통과 기준

수리 스테이지 하나는 아래를 모두 만족해야 한다.

1. 장면만 봐도 무엇이 고장났는지 알 수 있다.
2. 아이가 첫 조작 전에 긴 설명을 읽지 않아도 된다.
3. 핵심 개념이 부품과 슬롯 규칙으로 드러난다.
4. 오답 이유가 장치 반응으로 보인다.
5. 같은 능력을 다른 장치에 한 번 더 쓴다.
6. 키보드와 터치로 모두 완료할 수 있다.
7. 완료 뒤 수학 문장이 짧게 남는다.
