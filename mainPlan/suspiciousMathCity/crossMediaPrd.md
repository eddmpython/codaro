# 크로스미디어 PRD

## 1. 목적

`수상한 수학도시`는 앱 하나로 끝내지 않는다. 장기적으로 스토리북, 워크북, 조작 부록, 교사용 활동지로 확장 가능한 수학 세계관이어야 한다.

핵심 결정:

```text
앱, 도서, 워크북, 활동지는 서로 다른 줄거리를 만들지 않는다.
같은 Episode Pack에서 각 포맷을 파생한다.
```

## 2. 공통 원천 단위

하나의 에피소드는 `Episode Pack`으로 관리한다.

```ts
type EpisodePack = {
  episodeId: string;
  storyArc: StoryArc;
  worldMathBeats: WorldMathBeat[];
  abilityId: string;
  clueIds: string[];
  misconceptionIds: string[];
  representationLadder: RepresentationStep[];
  transferBeats: TransferBeat[];
  formatAdapters: {
    app: AppFormat;
    storybook: StorybookFormat;
    workbook: WorkbookFormat;
    teacherSheet: TeacherSheetFormat;
  };
};
```

`World Math Beat`는 앱 미션만의 단위가 아니라 모든 매체의 원천이다.

```text
이상현상
→ 단서 2개
→ 개념 행동
→ 시각 모델
→ 수학 문장
→ 오개념 회복
→ 세계 복구
→ 다음 단서
```

## 3. 포맷별 역할

| 포맷 | 핵심 경험 | 산출물 |
| --- | --- | --- |
| 앱 | 직접 조작하고 세계를 바꿈 | 지도, 장면, 능력, 단서장 |
| 스토리북 | 이상현상과 캐릭터 감정을 따라감 | 8-12쪽 이야기 |
| 워크북 | 조작을 종이 활동으로 반복 | 4쪽 활동 모듈 |
| 조작 부록 | 손으로 자르고 붙이고 놓음 | 타일, 카드, 분할판 |
| 교사용 활동지 | 발문과 협동 활동 | 15/30/45분 수업안 |
| 보호자 가이드 | 집에서 짧게 대화 | 오개념 설명, 말 걸기 질문 |

## 4. 포맷별 변환 매트릭스

| episodeId | 앱 조작 | 스토리북 장면 | 워크북 활동 | 교사용 발문 | 단서 | 능력 재사용 |
| --- | --- | --- | --- | --- | --- | --- |
| `clocktower-01` | 10분 조각 놓기 | 멈춘 시계탑과 버스 시간표 | 시계 빈칸 색칠 | 두 시각 중 무엇을 봐야 할까? | 0의 쪽지 | 버스 시간표 재방문 |
| `missing-zero-02` | 10, 100, 101 번호판 복구 | 0이 사라진 번호판 | 자리 카드 붙이기 | 0은 언제 자리를 지킬까? | 0의 자리표 | 숫자 버스 |
| `number-bus-03` | 번호 타일 배열 | 규칙을 잃은 버스 노선 | 점프 선분 그리기 | 앞뒤 번호 차이가 같을까? | 규칙 노선표 | 패턴숲 |
| `bakery-share-04` | 빵을 사람에게 공평 배분 | 접시가 다르게 채워진 빵집 | 빵 카드 나누기 | 같은 수로 나눴을까? | 공평 영수증 | 반쪽 빵집 |
| `bakery-fraction-05` | 케이크 같은 크기 자르기 | 원형 케이크와 손님 4명 | 1/2, 1/4 색칠 | 조각 수와 크기가 모두 같을까? | 나누기 칼 메모 | 지도 조각 |

## 5. 출판 가능 조건

한 에피소드는 아래로 자연스럽게 변환 가능해야 한다.

| 산출물 | 조건 |
| --- | --- |
| 앱 에피소드 | 6-8분 조작형 장면 |
| 스토리북 1장 | 8-12쪽, 찾아보기 컷 1개, 개념 한 문장 |
| 워크북 1단원 | 4쪽, 관찰-조작-수학 문장-전이 |
| 활동지 1장 | 15/30/45분 운영 가능 |
| 회상 카드 | 다음 날 1분 복습 가능 |

변환이 안 되는 에피소드는 앱에서도 확장하지 않는다.

## 6. 금지

- 앱 전용 보상 때문에 책 줄거리가 어긋나는 것.
- 책에서만 등장하는 핵심 설정이 앱에 없는 것.
- 워크북이 단순 계산 반복으로 변하는 것.
- 교사용 활동지가 앱 UI 설명서가 되는 것.
- 시즌 질문과 무관한 문항 묶음.

## 7. Product Core Slice 정의

이 프로젝트에서 첫 출시 단위는 버리는 MVP가 아니다. 이름은 `Product Core Slice`로 둔다.

첫 Product Core Slice:

```text
시계탑이 멈춘 날
```

완료 조건:

- 앱에서 조작형 장면으로 플레이 가능.
- 스토리북 1장으로 전환 가능.
- 워크북 4쪽 모듈로 전환 가능.
- 교사용 15분 활동으로 전환 가능.
- 능력 카드와 단서 카드가 후속 에피소드에서 재사용 가능.
- registry, progress, asset, accessibility, visual gate를 통과.
