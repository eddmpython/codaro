# 출판 포맷 매트릭스

## 1. 목적

이 문서는 앱 에피소드를 스토리북, 워크북, 조작 부록, 활동지로 변환하는 규칙이다. 출판물은 앱의 부록이 아니라 같은 원천에서 나온 다른 표현이어야 한다.

## 2. 워크북 4쪽 모듈

| 쪽 | 역할 | 구성 |
| --- | --- | --- |
| 1 | 장면 관찰 | 이상현상 그림, 단서 2개 찾기, 불필요 정보 1개 구분 |
| 2 | 조작 활동 | 오려 쓰는 타일, 선 긋기, 색칠, 붙이기 |
| 3 | 수학 문장 | 장면 -> 모델 -> 식 또는 말 |
| 4 | 전이와 회상 | 다른 물건에 같은 능력 쓰기, 오늘의 기억 문장 |

시계탑 예:

| 쪽 | 내용 |
| --- | --- |
| 1 | 2시 40분 시계와 3시 버스 시간표에서 단서 찾기 |
| 2 | 10분 타일 2개를 사이 칸에 놓기 |
| 3 | `2:40 + 20분 = 3:00`을 색 띠로 표현 |
| 4 | 1시 30분에서 2시까지 남은 시간 찾기 |

## 3. 스토리북 8-12쪽 구조

| 구간 | 쪽 | 역할 |
| --- | --- | --- |
| 사건 | 1-2 | 도시의 이상현상 |
| 관찰 | 3-4 | 노바의 질문과 단서 2개 |
| 시도 | 5-6 | 탐험가의 첫 조작과 오개념 반응 |
| 개념 순간 | 7-8 | 능력 사용과 세계 규칙 발견 |
| 복구 | 9-10 | 장소 변화 |
| 단서 | 11-12 | 제로의 쪽지와 다음 질문 |

책 속 질문은 문제지가 아니라 찾아보기 컷이어야 한다.

```text
시계와 시간표 중 무엇을 같이 봐야 할까?
```

## 4. 조작 부록

| 부록 | 쓰임 | 앱 능력 |
| --- | --- | --- |
| 10분 타일 | 시간 간격 놓기 | 시간 렌즈 |
| 번호 타일 | 규칙 배열 | 패턴 렌즈 |
| 자리값 카드 | 0과 자리 복구 | 자리 렌즈 |
| 케이크 분할판 | 같은 크기 나누기 | 나누기 칼 |
| 길이 막대 | 길이 합치기 | 거리 줄자 |
| 그래프 스티커 | 자료 분류와 비교 | 분석 망원경 |

부록은 정답을 맞히는 재료가 아니라 개념을 손으로 보는 도구다.

## 5. 교사용 활동지

| 시간 | 활동 | 목표 |
| --- | --- | --- |
| 15분 | 장면 관찰 + 조작 1개 | 개념 필요성 경험 |
| 30분 | 장면 관찰 + 조작 + 전이 | 같은 개념을 다른 장면에 적용 |
| 45분 | 이야기 읽기 + 팀 조작 + 발표 | 말로 설명하고 오개념 회복 |

시계탑 발문:

| 단계 | 발문 |
| --- | --- |
| 관찰 | 시계탑과 버스 시간표에서 각각 무엇을 알 수 있을까? |
| 조작 | 두 시각 사이에 10분 타일을 몇 개 놓을 수 있을까? |
| 개념 | 남은 시간은 어디에서 어디까지의 간격일까? |
| 전이 | 다른 버스도 같은 방법으로 찾을 수 있을까? |

## 6. 인쇄 품질 기준

| 항목 | 기준 |
| --- | --- |
| 글자 | 초등 저학년이 읽을 수 있는 크기와 줄간격 |
| 흑백 | 색 없이도 단서와 상태가 구분됨 |
| 여백 | 오려 붙이기와 필기 공간 충분 |
| 문장 | 한 문장 45자 이하 |
| 조작 | 가위 없이도 선 긋기/색칠 대체 가능 |
| 보호자 도움 | 정답보다 말 걸기 질문 제공 |

## 7. 출판 준비 판정

출판 준비는 다음을 통과해야 한다.

1. 앱 에피소드가 스토리북 8-12쪽으로 변환된다.
2. 워크북 4쪽 모듈이 나온다.
3. 조작 부록이 최소 1개 연결된다.
4. 교사용 15분 활동이 가능하다.
5. 흑백 출력에서도 단서와 상태가 구분된다.
6. 앱과 책의 줄거리가 어긋나지 않는다.
