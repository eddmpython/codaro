# 교과 연계 기준

## 1. 목적

이 문서는 `수상한 수학도시`의 개념 순서가 실제 초등 수학 발달과 어긋나지 않게 관리한다. 앱은 교과서처럼 보이면 안 되지만, 학습 구조는 교과 기준과 연결되어야 한다.

참고:

- 교육부 고시 제2022-33호 초중등학교 교육과정 총론 및 각론.
- 수학과 교육과정은 별책 8로 고시되어 있다.
- 초등학교 3, 4학년 적용은 2025년 3월 1일부터다.
- 출처: https://www.moe.go.kr/boardCnts/viewRenew.do?boardID=141&boardSeq=93458&lev=0&m=040401&opType=N&page=1&s=moe&searchType=null&statusYN=W

## 2. 데이터 필드

에피소드 데이터는 아래 필드를 가진다.

| 필드 | 목적 |
| --- | --- |
| `gradeBand` | `2-3`, `3-core`, `3-4` 같은 실제 난이도 표시 |
| `standardCodes` | 성취기준 코드 연결 |
| `prerequisiteChecks` | 시각 읽기, 건너세기, 전체-부분 등 진입 확인 |
| `readingDemand` | 문장 길이, 핵심 어휘, 불필요 정보 수 |
| `manipulativeModel` | 타일, 수직선, 분할 가이드, 그래프판 등 |
| `representationLadder` | 장면 -> 그림/모델 -> 수학 문장 -> 전이 |
| `misconceptionIds` | 수학 오개념 |
| `readingTrapIds` | 문제 상황을 잘못 읽는 독해 오개념 |
| `nearTransfer` | 같은 모델의 가까운 적용 |
| `farTransfer` | 다른 장면의 먼 적용 |
| `workbookPageType` | 인쇄 활동 확장 유형 |
| `bookSceneAnchor` | 도서 장면으로 확장할 핵심 컷 |

## 3. 발달 순서

| 순서 | 에피소드/장면 | 발달 초점 | 교과 연계 | 조작 활동 | 완료 근거 |
| --- | --- | --- | --- | --- | --- |
| 0 | 입장 진단 미니 장면 | 시각 읽기, 2씩 세기, 전체-부분 확인 | 초2 기초 | 시계 고르기, 번호 건너세기, 조각 비교 | 막히는 선행 감각을 벌점 없이 확인 |
| 1 | 시계탑 | 시각과 시간 간격 | 초2 시각, 초3-4 시간 계산 | 10분 타일을 사이 칸에 놓기 | 다른 시간표에도 같은 방식 적용 |
| 2 | 사라진 0 | 자리값, 0의 역할 | 4수01-01, 4수01-02 | 10, 100, 101 번호판 복구 | 10과 1이 다른 이유 설명 |
| 3 | 숫자 버스 | 같은 간격의 규칙 | 2수02, 4수02-01 | 번호 타일, 점프 선분 | 규칙을 말과 수식으로 표현 |
| 4 | 반쪽 빵집 A | 공평 배분 | 4수01-05 | 빵 여러 개를 사람에게 하나씩 나눔 | 같은 수로 나눔을 설명 |
| 5 | 반쪽 빵집 B | 등분할과 단위분수 | 4수01-09, 4수01-10 | 한 케이크를 같은 크기로 자르기 | `1/2`, `1/4`를 전체와 연결 |
| 6 | 길이 다리 | 단위 반복과 길이 합 | 4수03-15, 4수03-16 | 길이 막대 이어 붙이기 | 눈대중이 아니라 단위 수로 판단 |
| 7 | 도형 공방 | 모양 분류, 회전 불변성 | 2수03, 4수03 | 도형 돌리기, 분류판 | 방향이 바뀌어도 같은 도형 판별 |
| 8 | 데이터 항구 | 표, 그래프 해석 | 2수04, 4수04 | 사건 카드 분류, 막대그래프 만들기 | 자료에 맞는 그래프 선택 |

## 4. 오개념과 독해 함정

| 개념 | 수학 오개념 | 독해 함정 | 회복 질문 |
| --- | --- | --- | --- |
| 시간 간격 | 3시까지 무조건 60분 | 현재 시각을 읽지 않음 | 지금 분침은 어디에서 시작하지? |
| 0과 자리값 | 10에서 0을 지워도 비슷함 | 10번과 1번을 같은 번호로 읽음 | 0이 없어지면 숫자의 자리가 어떻게 바뀌지? |
| 규칙 | 항상 1씩 커짐 | 앞뒤 번호를 비교하지 않음 | 앞 번호와 뒤 번호 차이가 같아? |
| 공평 배분 | 조각 수만 같으면 공평 | 손님 수와 조각 크기를 따로 봄 | 개수 말고 크기도 같아? |
| 분수 | 분모가 크면 조각도 큼 | 전체가 같다는 조건을 놓침 | 전체가 같을 때 몇 조각으로 나눴지? |
| 길이 | 눈으로 길어 보이면 더 김 | 시작점을 맞추지 않음 | 출발선을 맞췄어? |
| 도형 | 돌리면 다른 도형 | 방향과 모양을 혼동 | 변의 수와 꼭짓점 수는 그대로야? |
| 그래프 | 막대가 예쁘면 맞음 | 범주 중복을 놓침 | 한 물건이 두 칸에 들어갔어? |

## 5. 독해 결합 체크리스트

- 문제 전에 무엇이 이상한지 말하게 한다.
- 필요한 단서 2개와 불필요한 단서 1개를 함께 둔다.
- 새 용어는 장면 조작 뒤에 나온다.
- `시각/시간`, `전체/부분`, `규칙/간격`, `공평/같은 크기`를 어휘 카드로 남긴다.
- 오답이 계산 오류인지, 단서 읽기 오류인지 분리한다.
- 완료 문장은 `나는 ___을 보고 ___을 고쳤다` 형태로 말하게 한다.
