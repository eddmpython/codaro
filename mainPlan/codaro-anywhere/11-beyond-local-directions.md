# 11. Beyond-Local Directions - 로컬급을 넘보는 발명급 방향 검토

상태: 탐색 + 부분 실측 v0.1 (2026-07-11). 목표(사용자): 브라우저-서버([10]) 다음으로, 더 혁신적이고 진짜 로컬급이거나 로컬을 초월하는 방향 검토. 발명급.

---

## 0. 출발점: 유일한 약점을 뒤집기
[10]까지 확정: 서버/자동화/파이썬 로직은 브라우저가 로컬급(3.14 인터프리터는 로컬 3.12보다 빠름). **유일한 진짜 약점 = numpy/BLAS 대규모 수치 ~86배 느림**(WASM 단일스레드+128bit SIMD vs 네이티브 멀티코어+AVX). 발명급 방향의 1순위 = 이 약점을 강점으로 뒤집기.

## 1. WebGPU 수치 가속 (로컬 CPU numpy를 이기는 각도)
- 메커니즘: numpy/텐서 연산(matmul/conv/reduction)을 WebGPU compute shader(WGSL)로 GPU에서 실행. Pyodide 파이썬 배열 -> GPU 버퍼 -> 커널 -> 결과 회수. Pyodide는 JS interop로 navigator.gpu 호출 가능.
- 왜 로컬 초월 가능: **로컬 파이썬 numpy는 GPU를 기본으로 안 쓴다(CPU BLAS).** 브라우저는 WebGPU로 GPU에 직접 접근. 큰 행렬곱/딥러닝 추론은 GPU가 CPU를 수십~수백배 이길 수 있어, 브라우저가 로컬 CPU numpy를 이길 여지.
- **실측 상태(정직)**: WebGPU는 2023년부터 정식 API(Chrome 113+). 그러나 **이 headless 환경에선 검증 불가** - GPU 어댑터 없음, 소프트웨어 폴백(swiftshader) 어댑터도 안 잡힘. 즉 "GPU로 numpy 이긴다"는 속도 숫자를 여기서 실증 못 함. 사용자 실제 PC(GPU 있음)에선 동작 예상. 선례: tensorflow.js/ONNX Runtime Web/wonnx가 WebGPU로 GPU 추론 실현.
- 한계: WebGPU 배선 복잡(버퍼/커널/동기화), f32 위주(f64 제한), 작은 연산은 전송 오버헤드로 손해, 브라우저/드라이버 편차. WebNN API(OS 신경망 가속)는 더 이르지만 관찰 대상.

## 2. 실행 환경을 값으로 (로컬이 원리적으로 못 하는 것)
- 메커니즘: **Pyodide 공식 `makeMemorySnapshot` API 존재(실측 확인)** + 검증된 HEAPU8 커스텀 스냅샷([08 계열]). 프레임워크·패키지·데이터까지 로드된 인터프리터 상태를 통째로 스냅샷 값으로 만들어 (a) 즉시 부팅(재-import 스킵), (b) 다른 인스턴스/기기로 전송, (c) fork/시간여행.
- 왜 로컬 초월: **로컬 파이썬은 실행 중 인터프리터를 통째로 스냅샷해 다른 곳에 복원할 수 없다**(Windows fork 없음, CRIU는 리눅스 한정 무거움). 브라우저는 WASM 선형메모리가 ArrayBuffer라 통째 복사/복원이 자명. "실행 환경 = 포터블 값"은 로컬이 못 하는 것.
- 실측 상태: 커스텀 HEAPU8 스냅샷/복원 PASS(리액티브 4배, page-diff 10~14MB, [08 §3.5]). 공식 makeMemorySnapshot 존재 확인. 크로스-인스턴스 전송의 실용 병목 = 크기(전체 ~360MB, page-diff로 셀당 10~14MB).
- 한계: 크기(URL 공유엔 큼, 압축/증분 필요), JS측 핸들(열린 파일/소켓/DOM)은 스냅샷 안 됨(순수 파이썬 상태 한정), emval 경계(별도 연구).

## 3. (토론 랭킹 대기)
전문가 토론(WebGPU가속/브라우저고유자원/상주데몬/초월발명가 4렌즈)이 진행 중. 완료되면 추가 방향(WebCodecs 하드웨어 비디오, Web Audio DSP, WebNN, OPFS 진짜 로컬DB, 상주 데몬, 즉시공유, 샌드박스 안전성)과 랭킹을 여기 통합.

## 4. 잠정 판정
- **1순위 WebGPU 수치 가속**: 유일 약점(numpy 86배)을 뒤집는 유일한 각도. 발명성·초월성 높으나 이 환경에서 실측 불가(GPU 필요). 사용자 실PC 실측이 다음.
- **2순위 실행환경=값**: 로컬이 원리적으로 못 하는 것, 공식+커스텀 둘 다 검증. 크기가 실용 병목.
- 둘 다 검증된 자산([08]자족티어·[10]브라우저서버·스냅샷)과 결합. 서버가 로컬급 + WebGPU로 수치까지 로컬 초월 + 환경을 값으로 공유 = "브라우저가 로컬을 넘는" 그림.
