# Five Minute Quickstart

## 0:00-0:30 — What This Is

Codaro는 Python 학습, 코드 실행, 개인 자동화를 한 화면에서 이어 주는 local-first 스튜디오다.

## 0:30-1:30 — Run A Demo Script

```powershell
uv run python -X utf8 demos/publicLaunch/expenseSummaryDemo.py
```

확인할 것:

- CSV를 직접 읽는다.
- 카테고리별 합계가 나온다.
- 가장 큰 지출 카테고리가 표시된다.
- 같은 코드를 Codaro 셀에 붙여 학습 자료로 바꿀 수 있다.

## 1:30-2:30 — See A Safe Automation Plan

```powershell
uv run python -X utf8 demos/publicLaunch/fileOrganizerDemo.py
```

확인할 것:

- 파일을 실제로 옮기지 않는다.
- 확장자별 이동 계획만 보여준다.
- 사용자가 확인한 뒤 자동화 task로 키울 수 있다.
