# Video Storyboard

목표 길이: 90초. 핵심은 “설명”이 아니라 화면에서 바로 이해되는 장면이다.

## Scene 1 — Hook

시간: 0-10초

화면:

- Codaro README 또는 landing 첫 화면
- 한 줄 문구: Python 학습, 실행, 자동화를 한 화면에서

말:

> Python을 배우다가 바로 실행하고, 반복 작업은 자동화로 키우는 도구를 만들고 있습니다.

## Scene 2 — Learning

시간: 10-30초

화면:

- Python 30일 과정 Day 01
- 학습 카드와 코드 셀
- 실행 결과

말:

> 처음에는 학습 카드로 시작합니다. 설명을 보고, 코드를 실행하고, 값을 바꿔 보면서 배웁니다.

## Scene 3 — Run Real Data

시간: 30-50초

화면:

- `expenseSummaryDemo.py`
- CSV 입력
- 카테고리별 합계 출력

말:

> 학습용 예제에서 끝내지 않고, 실제 CSV 같은 작은 데이터를 바로 다룹니다.

## Scene 4 — Automation Plan

시간: 50-70초

화면:

- `fileOrganizerDemo.py`
- dry-run 이동 계획
- 안전하게 “계획만 보기”

말:

> 반복 작업은 바로 실행하지 않고, 먼저 안전한 계획으로 확인합니다. 그 다음 task와 workflow로 키울 수 있습니다.

## Scene 5 — Trust

시간: 70-82초

화면:

- `publicReadinessChecklist.md`
- `SECURITY.md`
- `PRIVACY.md`
- gate command output 일부

말:

> 공개용으로 보안, 개인정보, 지원, 검증 gate도 같이 준비했습니다.

## Scene 6 — Close

시간: 82-90초

화면:

- README quickstart 링크
- GitHub repository

말:

> Codaro는 무료로 공개합니다. Python을 배우고 자동화까지 가고 싶다면 바로 실행해 보세요.
