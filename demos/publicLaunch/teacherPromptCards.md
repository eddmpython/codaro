# Teacher Prompt Cards

Codaro 안에서 바로 써볼 수 있는 prompt 카드다. provider가 연결되지 않아도 기본 curriculum과 demo script로 같은 흐름을 수동 실행할 수 있어야 한다.

## Learning Prompt

```text
Python 입문자 기준으로 CSV 파일을 읽고 카테고리별 합계를 구하는 수업을 만들어줘.
설명, 실행 코드, 연습 문제, 확인 질문 순서로 구성해줘.
```

## Automation Prompt

```text
다운로드 폴더 파일을 확장자별로 정리하는 자동화 계획을 만들어줘.
처음에는 dry-run만 하고 실제 이동은 사용자가 확인한 뒤에 하게 해줘.
```

## Debug Prompt

```text
이 코드가 파일을 못 찾을 때 사용자가 이해할 수 있는 진단 메시지와 복구 단계를 만들어줘.
```

## Curriculum Prompt

```text
Python 30일 과정 중 파일 입출력 다음에 넣을 짧은 실습을 만들어줘.
CSV 읽기, 합계 계산, 결과 해석을 포함해줘.
```
