# 비전 응용 (visionApps) PRD

## 트랙 목적

비전 기법을 실제 업무 결과물로 연결하는 트랙. 문서 스캐너, OCR, 얼굴 익명화, 이미지 중복 검출 — 모두 학습자가 자기 폴더의 사진과 영상에 적용해 즉시 가치를 만들 수 있는 도구들이다.

이 트랙은 visionBasics, opencv, visionFeatures, (선택적으로) deepVision 트랙의 결과를 모아 산출물로 변환하는 응용 트랙이다.

## 학습 후 도달 상태

- 사진 한 장으로 문서 스캐너를 만들 수 있다.
- QR/바코드 디코더를 한 줄로 호출할 수 있다.
- pytesseract / easyocr 로 OCR을 실행할 수 있다.
- 얼굴 박스를 검출하고 모자이크를 적용할 수 있다.
- imagehash로 폴더 안 중복 사진을 찾을 수 있다.
- 영상에서 자동으로 대표 썸네일 N장을 뽑을 수 있다.
- 화면 캡처를 비전 입력으로 자동 처리할 수 있다.
- 위 도구들을 묶은 자동 사진 정리기를 만들 수 있다.

## 데이터 소스

각 강의가 외부 의존을 최소화하도록 다음 패턴을 따른다.

1. **이미지 합성**: sklearn china/flower + cv2로 사각형·텍스트·QR을 그려 학습 입력 생성
2. **임시 폴더**: `Path(tempfile.gettempdir()) / 'codaro_apps'` 에 학습용 폴더 트리를 만든다
3. **영상**: visionFeatures 6강에서 만든 codaro_demo.mp4 재사용

## 사용 라이브러리

- `opencv-python` (필수)
- `numpy`, `matplotlib`
- `pytesseract` (3강 OCR) — 시스템에 tesseract가 설치되어 있어야 함
- `easyocr` (4강 한국어 OCR) — 첫 호출 시 모델 다운로드 (~ 100MB)
- `pyzbar` (2강 바코드) — libzbar 시스템 라이브러리 필요
- `face_recognition` (6강) — dlib 의존성, 설치 까다로움
- `imagehash`, `pillow` (7강)
- `mss` (9강) — 화면 캡처

학습자에게 라이브러리 설치 안내를 각 강의 첫 셀에 명시한다. tesseract/zbar/dlib처럼 시스템 라이브러리 의존이 있는 경우는 강의 첫 셀의 tip에서 명확히 안내한다.

## 10개 강의 설계

| # | 제목 | 핵심 도구 | 산출물 | 난이도 |
|---|---|---|---|---|
| 01 | 문서 스캐너 | opencv 호모그래피 | 사진 → 정렬된 스캔 이미지 | ⭐⭐⭐ |
| 02 | QR/바코드 리더 | pyzbar | 이미지 → 디코드 결과 | ⭐⭐ |
| 03 | OCR 기초 | pytesseract | 이미지 → 텍스트 | ⭐⭐⭐ |
| 04 | 한국어 OCR | easyocr | 한국어 사진 → 텍스트 | ⭐⭐⭐ |
| 05 | 얼굴 검출과 모자이크 | opencv haar | 얼굴 박스 + 모자이크 | ⭐⭐⭐ |
| 06 | 얼굴 임베딩과 동일인 판별 | face_recognition | 두 사진 일치 여부 | ⭐⭐⭐⭐ |
| 07 | 이미지 중복 검출 | imagehash | 폴더 → 중복 그룹 | ⭐⭐⭐ |
| 08 | 영상 자동 썸네일 | opencv + 프레임 차분 | 영상 → 대표 N장 | ⭐⭐⭐ |
| 09 | 화면 캡처 분석 | mss + opencv | 화면 영역 → 분석 | ⭐⭐⭐ |
| 10 | 종합: 사진 자동 정리기 | 위 도구 조합 | 폴더 입력 → 분류된 결과 | ⭐⭐⭐⭐ |

## 강의 작성 원칙

1. 시스템 라이브러리 의존이 있는 도구는 강의 첫 셀에 install 가이드와 fallback 경로를 안내한다.
2. 임시 폴더는 항상 `Path(tempfile.gettempdir()) / 'codaro_apps_<강의>'` 패턴으로 격리한다.
3. 외부 데이터 다운로드 없이 학습이 완결되도록 모든 입력은 합성한다.
4. 산출물은 노트북 셀의 마지막 평가로 즉시 확인 가능한 형태로 만든다.
5. 변수 재할당 금지.

## 의존 트랙

- 1강 문서 스캐너: visionFeatures 4강(호모그래피) 의 패턴 활용
- 5강 얼굴 모자이크: visionBasics 5강(슬라이싱·모자이크) 의 패턴 활용
- 8강 영상 썸네일: visionFeatures 6~7강(비디오 + 차분) 활용
- 10강 종합: deepVision 10강의 autoTag 함수까지 옵션으로 활용 가능
