# Code of Conduct

Codaro 커뮤니티와 기여 공간은 학습자, 유지보수자, 보안 제보자가 안전하게 기술 논의를 이어갈 수 있어야 합니다.

## Expected Behavior

- 사람을 공격하지 않고 문제, 재현 단계, 근거를 다룹니다.
- 초보 학습자에게 필요한 맥락을 제공합니다.
- 보안 취약점과 개인정보 문제는 공개 확산보다 책임 있는 보고를 우선합니다.
- 상업적 사용, 브랜드 사용, 라이선스 경계를 존중합니다.

## Unacceptable Behavior

- 괴롭힘, 차별, 위협, 모욕
- 개인 정보나 credential 공개
- 취약점 악용 방법의 공개 확산
- 반복적인 off-topic 홍보나 스팸
- 유지보수자에게 비공개 정보를 공개하라고 압박하는 행위

## Enforcement

유지보수자는 issue/comment 삭제, 경고, 일시 차단, 영구 차단, 보안 신고 경로 전환을 할 수 있습니다. 심각한 보안/개인정보 사안은 공개 토론을 중단하고 `SECURITY.md` 절차로 옮깁니다.
