# 세금계산서 검증 PRD

## 개요
홈택스 세금계산서와 ERP 세금계산서를 대사하여 불부합 내역을 추출하는 부가세 검증 자동화 도구

## 문제 정의

### 현재 상황 (수작업)
1. 홈택스에서 세금계산서 엑셀 다운로드
2. ERP에서 부가세 자료 엑셀 내보내기
3. 수작업으로 사업자번호, 금액 대조
4. 불일치 건 수동으로 찾아서 엑셀 정리
5. **문제점**: 시간이 오래 걸리고, 실수 발생 가능, 중복 검증 어려움

### 자동화 후
1. 홈택스 엑셀 + ERP 엑셀 업로드
2. 컬럼 매핑 (어떤 열이 사업자번호/금액인지 선택)
3. 자동 대사 실행
4. 3개 시트 다운로드: ① 병합원본 ② 매칭결과 ③ 불부합결과
5. **효과**: 3분 작업 → 30초, 정확도 100%, 재현 가능

## 핵심 기능

### 1. 파일 업로드
- **홈택스 세금계산서 엑셀** (매출 또는 매입)
- **ERP 세금계산서 엑셀** (매출 또는 매입)
- 매출/매입 구분 라디오 버튼

### 2. 컬럼 매핑 UI
#### 왜 필요한가?
- 홈택스: `공급받는자사업자등록번호`, `공급가액`, `세액`
- ERP: 회사마다 다름 (`사업자번호`, `공급가`, `부가세금액` 등)
- **사용자가 직접 매핑**해야 정확한 대사 가능

#### UI 구성

#### 자동 추천 (Phase 2)
- 컬럼명에 "사업자", "등록번호", "biz", "no" 포함 → 사업자번호로 추천
- "공급", "가액", "금액", "supply" 포함 → 공급가액으로 추천
- "세액", "부가세", "tax", "vat" 포함 → 세액으로 추천

### 3. 데이터 전처리
1. **사업자번호 정규화**: 하이픈 제거 (`123-45-67890` → `1234567890`)
2. **정렬**: 사업자번호 → 공급가액 → 세액 순
3. **KEY 컬럼 생성**:
   - `KEY_사업자등록번호`
   - `KEY_공급가액`
   - `KEY_세액`
   - `KEY_중복체크` (같은 사업자+금액이 여러 건일 경우 1, 2, 3...)

### 4. 병합 및 대사
- **pandas merge (outer join)**
- 4개 KEY로 매칭: 사업자번호 + 공급가액 + 세액 + 중복체크
- 양쪽에 모두 있으면 → **매칭 성공**
- 한쪽만 있으면 → **불부합 건**

### 5. 결과 다운로드 (3시트 엑셀)
#### Sheet 1: `병합원본`
- merge 전 HTS, ERP 각각의 원본 데이터
- 사용자가 원본 확인 가능

#### Sheet 2: `매칭결과`
- merge 결과 전체
- 양쪽 컬럼이 모두 표시됨
- 사용자가 매칭 과정 검증 가능

#### Sheet 3: `불부합결과` ⭐
- **ERP에만 있거나 HTS에만 있는 건**
- 필터링 조건: `isnull()`로 한쪽 사업자번호가 비어있는 행
- **이것이 실무에서 가장 중요한 시트**

## 기술 스택

### Python 라이브러리
- `pandas`: 엑셀 읽기/쓰기, DataFrame 조작, merge
- `openpyxl`: 엑셀 파일 쓰기 엔진
- `marimo`: UI (라디오, 파일 업로드, 드롭다운, 테이블)
- `BytesIO`: 메모리 기반 파일 처리

### 핵심 로직
```python
# 1. 파일 읽기
hts = pd.read_excel(BytesIO(htsFile.contents(0)), header=헤더행번호)
erp = pd.read_excel(BytesIO(erpFile.contents(0)), header=헤더행번호)

# 2. 컬럼 선택 (사용자 매핑)
HTS = hts[[사업자번호컬럼, 공급가액컬럼, 세액컬럼, ...]].copy()
ERP = erp[[사업자번호컬럼, 공급가액컬럼, 세액컬럼, ...]].copy()

# 3. 전처리
HTS['사업자번호'] = HTS['사업자번호'].str.replace('-', '', regex=False)
HTS = HTS.sort_values(...)
HTS['KEY_사업자등록번호'] = HTS['사업자번호']
HTS['KEY_중복체크'] = HTS.groupby(['KEY_사업자등록번호', 'KEY_공급가액']).cumcount() + 1

# 4. 병합
merged = pd.merge(ERP, HTS, on=[KEY 컬럼들], how='outer')

# 5. 불부합 추출
mismatch = merged.loc[
    (merged['ERP_사업자번호'].isnull()) | 
    (merged['HTS_사업자번호'].isnull())
]

# 6. 엑셀 저장 (3시트)
with pd.ExcelWriter(buffer, engine='openpyxl') as writer:
    HTS.to_excel(writer, sheet_name='홈택스원본', index=False)
    ERP.to_excel(writer, sheet_name='ERP원본', index=False)
    merged.to_excel(writer, sheet_name='매칭결과', index=False)
    mismatch.to_excel(writer, sheet_name='불부합결과', index=False)
```

```python
# Notion 학습 자료에서 추출한 코드

def readFiles(pattern, header=None):
    """여러 파일 읽기 (glob 사용, 로컬 환경용)"""
    files = glob.glob(pattern)
    data = [pd.read_excel(f, header=header) for f in files]
    return pd.concat(data, ignore_index=True)

def readAndProcessFile(df, prefix, columns, keyColumns):
    """단일 DataFrame 전처리"""
    df = df[columns].copy()
    df.columns = f'{prefix}_' + df.columns
    df[f'{prefix}_{keyColumns[0]}'] = (
        df[f'{prefix}_{keyColumns[0]}'].astype(str).str.replace('-', '', regex=False)
    )
    df = df.sort_values(
        by=[f'{prefix}_{keyColumns[0]}', f'{prefix}_{keyColumns[1]}', f'{prefix}_{keyColumns[2]}'],
        ascending=[True, True, True]
    )
    df = df.assign(
        KEY_사업자등록번호=df[f'{prefix}_{keyColumns[0]}'],
        KEY_공급가액=df[f'{prefix}_{keyColumns[1]}'],
        KEY_세액=df[f'{prefix}_{keyColumns[2]}']
    )
    df['KEY_중복체크'] = df.groupby(['KEY_사업자등록번호', 'KEY_공급가액']).cumcount() + 1
    return df
```
