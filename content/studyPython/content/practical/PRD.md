# 실전파이썬 PRD (Product Requirements Document)

## 개요
실전파이썬은 실무에서 바로 사용할 수 있는 작은 단위의 마리모 앱을 학습 콘텐츠와 함께 제공하는 섹션이다.

## 핵심 가치
1. **실용성**: 실제 업무에서 바로 사용 가능한 도구
2. **학습성**: 도구의 원리와 구현 방법을 함께 학습
3. **즉시성**: 마운트된 마리모 앱으로 즉시 사용 가능

## 사용자 시나리오

### 시나리오 1: 학습자
1. 실전파이썬 목록에서 "엑셀파일 병합" 클릭
2. study.yaml 기반 학습 페이지에서 원리 학습
3. "앱 사용하기" 버튼으로 마운트된 마리모 앱 이동
4. 실제 파일로 기능 테스트

### 시나리오 2: 도구 사용자
1. 마리모 앱 목록에서 "엑셀파일 병합" 앱 직접 사용
2. "이거 어떻게 만들었지?" → "학습하기" 링크 클릭
3. study.yaml 학습 페이지로 이동하여 구현 원리 학습

## 폴더 구조

```
content/practical/
├── PRD.md                    # 이 문서
├── PRINCIPLES.md             # YAML 작성 원칙
├── 00_실전파이썬소개.yaml     # 실전파이썬 소개 페이지
├── 엑셀파일병합/
│   ├── study.yaml            # 학습 콘텐츠
│   └── note.py               # 마리모 앱 (마운트용)
├── PDF병합/
│   ├── study.yaml
│   └── note.py
└── 이미지일괄변환/
    ├── study.yaml
    └── note.py
```

## 기술 구현

### 1. 라우팅 구조
- `/study/실전파이썬/{contentId}` - 학습 페이지 (study.yaml 렌더링)
- `/app/practical/{appId}` - 마리모 앱 (note.py 마운트)

### 2. MarimoMounter 클래스
- FastAPI에 마리모 앱을 마운트하는 유틸리티
- `content/practical/*/note.py` 파일을 자동 탐색하여 마운트
- URL: `/app/practical/{폴더명}`

### 3. study.yaml 특수 블록
- `marimoApp` 블록: 해당 콘텐츠의 마리모 앱 링크 버튼
- `appPreview` 블록: 마리모 앱 미리보기 iframe

## 콘텐츠 기획 (초기 목록)

### 파일 처리
1. 엑셀파일병합 - 여러 엑셀 파일을 하나로 병합
2. PDF병합 - 여러 PDF를 하나로 합치기
3. 이미지일괄변환 - PNG/JPG 일괄 변환 및 리사이즈
4. CSV인코딩변환 - EUC-KR ↔ UTF-8 변환

### 데이터 처리
5. JSON정리기 - JSON 포맷팅 및 검증
6. 중복데이터제거 - 엑셀/CSV 중복 행 제거
7. 날짜형식변환 - 다양한 날짜 포맷 통일

### 텍스트 처리
8. 정규식테스터 - 정규표현식 실시간 테스트
9. 텍스트비교기 - 두 텍스트 차이점 비교
