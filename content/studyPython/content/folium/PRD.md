# Folium 카테고리 PRD

## 개요
Folium은 Leaflet.js 기반의 인터랙티브 지도 시각화 라이브러리입니다.
파이썬으로 지도를 생성하고 마커, 폴리곤, 히트맵 등을 추가합니다.

## 데이터 특성
- **좌표 데이터**: 위도/경도만 있으면 지도 표시 가능
- **외부 데이터 불필요**: 좌표를 직접 입력하거나 간단한 리스트 사용
- **GeoJSON**: 지역 경계 데이터 (온라인 소스 활용)

## Pyodide/Marimo 환경
- folium은 순수 파이썬으로 pyodide에서 지원
- `mo.iframe(m._repr_html_())` 로 지도 렌더링
- JavaScript가 포함된 HTML이므로 mo.iframe() 필수

## 핵심 개념 (A-L, 12개)

| ID | 개념 | 설명 |
|----|------|------|
| A | 기본 지도 | Map(), location, zoom_start, tiles |
| B | 마커 | Marker(), popup, tooltip, icon |
| C | 원/사각형 | Circle(), CircleMarker(), Rectangle() |
| D | 폴리라인/폴리곤 | PolyLine(), Polygon(), 경로/영역 |
| E | 팝업/툴팁 | Popup(), Tooltip(), HTML 내용 |
| F | 타일 레이어 | TileLayer(), 지도 스타일 변경 |
| G | 레이어 컨트롤 | LayerControl(), FeatureGroup() |
| H | GeoJSON | GeoJson(), 지역 경계, choropleth |
| I | 히트맵 | HeatMap(), 밀도 시각화 |
| J | 마커 클러스터 | MarkerCluster(), 대량 마커 |
| K | 커스텀 아이콘 | Icon(), DivIcon(), 사용자 정의 |
| L | 인터랙션 | ClickForMarker(), 줌/패닝 |

**내용**:

## 변수명 규칙

### 지도 변수
- 기본: `m`, `map1`, `seoulMap`
- 미션별: `mapA`, `mapB`, `tourMap`

### 마커/레이어
- 마커: `marker`, `mk`, `cafeMarker`
- 그룹: `group`, `fg`, `cafeGroup`
- 레이어: `layer`, `tileLayer`, `geoLayer`

### 좌표
- 위치: `loc`, `center`, `seoulCenter`
- 좌표리스트: `coords`, `points`, `route`

### 데이터
- GeoJSON: `geojson`, `geoData`, `seoulGeo`
- 히트맵: `heatData`, `densityData`

## Marimo 출력 규칙

**반드시 `mo.iframe()` 사용:**
```python
import folium
import marimo as mo

m = folium.Map(location=[37.5665, 126.9780], zoom_start=12)
mo.iframe(m._repr_html_())
```

- 모든 지도 출력은 `mo.iframe(m._repr_html_())` 필수
