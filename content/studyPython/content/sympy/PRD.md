# SymPy 카테고리 PRD

## 개요
SymPy는 파이썬 기호 수학 라이브러리입니다. 숫자 계산이 아닌 **수식 자체**를 다룹니다.
데이터셋 대신 **수학 문제/수식**이 프로젝트 소재입니다.

## Pyodide 환경
- SymPy는 순수 파이썬으로 pyodide에서 완벽 지원
- import sympy로 바로 사용 가능
- matplotlib과 연동하여 수식 시각화 가능

## 핵심 개념 (A-L, 12개)

| ID | 개념 | 설명 |
|----|------|------|
| A | 기호 정의 | symbols(), Symbol(), 여러 기호 동시 정의 |
| B | 수식 생성 | 사칙연산, 거듭제곱, 분수, 수식 표현 |
| C | 수식 정리 | simplify(), expand(), factor(), collect() |
| D | 대입과 평가 | subs(), evalf(), N() |
| E | 방정식 풀이 | solve(), solveset(), Eq() |
| F | 미분 | diff(), Derivative(), 편미분 |
| G | 적분 | integrate(), 정적분, 부정적분 |
| H | 극한 | limit(), 좌극한, 우극한 |
| I | 급수와 합 | summation(), Sum(), series() |
| J | 행렬 | Matrix(), 행렬연산, 역행렬, 행렬식 |
| K | 삼각함수/지수 | sin, cos, tan, exp, log, 항등식 |
| L | 시각화 | plot(), plot3d(), matplotlib 연동 |

## 10개 프로젝트

### 01_기초수식계산 (입문) ⭐
**목표**: 기호 정의하고 수식 만들어서 정리하기
**개념**: A(기호정의), B(수식생성), C(수식정리)
**내용**:
- symbols()로 x, y, z 정의
- 다항식 수식 생성
- expand(), factor(), simplify() 적용
- collect()로 특정 변수로 정리

### 02_대입과계산 (입문) ⭐
**목표**: 수식에 값 대입하고 수치 계산하기
**개념**: A, B, C + D(대입평가)
**내용**:
- subs()로 값 대입
- evalf()로 소수점 계산
- 여러 변수 동시 대입
- 수식 간소화 후 대입

### 03_일차방정식풀이 (기초) ⭐⭐
**목표**: 다양한 방정식 풀기
**개념**: A, B, D + E(방정식풀이)
**내용**:
- Eq()로 방정식 정의
- solve()로 풀이
- 연립방정식 풀이
- solveset()으로 해집합

### 04_미분기초 (기초) ⭐⭐
**목표**: 함수 미분하고 접선 구하기
**개념**: A, B, C, D, E + F(미분)
**내용**:
- diff()로 미분
- 고차 미분
- 편미분
- 접선의 방정식 구하기

### 05_적분기초 (기초) ⭐⭐
**목표**: 부정적분과 정적분 계산
**개념**: A, B, C, D, F + G(적분)
**내용**:
- integrate()로 부정적분
- 정적분 계산
- 치환적분, 부분적분
- 넓이 계산

### 06_극한과급수 (중급) ⭐⭐⭐
**목표**: 극한값 계산하고 급수 전개하기
**개념**: A, B, C, D, F, G + H(극한), I(급수)
**내용**:
- limit()로 극한 계산
- 좌극한, 우극한
- series()로 테일러 급수
- summation()으로 급수 합

### 07_삼각함수활용 (중급) ⭐⭐⭐
**목표**: 삼각함수 항등식과 방정식 풀기
**개념**: A, B, C, D, E, F, G + K(삼각함수)
**내용**:
- sin, cos, tan 수식
- 삼각함수 항등식 증명
- 삼각방정식 풀이
- 삼각함수 미적분

### 08_행렬연산 (중급) ⭐⭐⭐
**목표**: 행렬 연산과 선형대수 계산
**개념**: A, B, C, D, E + J(행렬)
**내용**:
- Matrix() 생성
- 행렬 사칙연산
- 역행렬, 행렬식
- 고유값, 고유벡터
- 연립방정식 행렬 풀이

### 09_수식시각화 (심화) ⭐⭐⭐⭐
**목표**: 함수 그래프와 수식 시각화
**개념**: A, B, C, D, E, F, G, K + L(시각화)
**내용**:
- plot()으로 2D 그래프
- 여러 함수 동시 그리기
- matplotlib 연동
- 미분/적분 결과 시각화

### 10_종합수학문제 (심화) ⭐⭐⭐⭐
**목표**: 실제 수학 문제 종합 풀이
**개념**: A-L 모든 개념 종합
**내용**:
- 최적화 문제 (미분 활용)
- 넓이/부피 계산 (적분 활용)
- 물리 문제 (미분방정식)
- 경제학 문제 (한계비용/수익)

## 변수명 규칙

### 기호 변수
- 단일 문자: `x`, `y`, `z`, `t`, `n`, `a`, `b`, `c`
- 그리스 문자: `alpha`, `beta`, `theta`

### 수식 변수
- 표현식: `expr`, `eq`, `poly`
- 결과: `result`, `sol`, `ans`
- 미션별: `expr1`, `expr2`, `sol1`, `sol2`
- 의미있는 이름: `velocity`, `area`, `cost`

### 행렬 변수
- 행렬: `mat`, `matA`, `matB`
- 벡터: `vec`, `vecX`, `vecB`

### 시각화 변수
- 플롯: `plot1`, `plot2`, `figFunc`
- matplotlib: `fig`, `ax`, `figArea`

## 프로젝트 구조

각 프로젝트는 다음 구조를 따름:
1. 라이브러리 import
2. 기호 정의
3. 수식 생성 및 조작
4. 계산/풀이
5. 결과 확인 (시각화 포함 시)
6. 실습 미션 2개

## 특이사항

- 데이터셋 불필요 (수학 수식이 소재)
- print() 대신 표현식 자체 출력
- sympy는 자동으로 LaTeX 형태로 수식 렌더링
- matplotlib 연동 시 한글 폰트 불필요 (수식만 표시)
