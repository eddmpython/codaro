# Python 첫 실행 공유 팩

이 pack은 Codaro 공유 팩 설치 흐름을 검증하기 위한 공식 예제입니다.

- `codaroPack.yaml`을 Codaro의 공유 팩 화면에 붙여넣고 검사합니다.
- 검사 결과를 확인한 뒤 설치합니다.
- 설치된 커리큘럼은 나만의 커리큘럼으로 열 수 있습니다.
- 자동화 recipe는 dry-run 상태로 태스크에 등록할 수 있습니다.
