## 정체성

Codaro는 코드가 인터페이스가 되는 **개인 자동화 스튜디오**. 제품 표면은 채팅, 에디터, 커리큘럼, 자동화 네 가지이고, 리포트는 자동화/태스크 실행 결과의 산출물로 다룬다.

## 절대 게이트 (요약, 위반 시 즉시 차단)

- Python 3.12+ + `uv` 전용. `pip` 직접 사용 금지. 테스트: `uv run python -X utf8 tests/run.py preflight`. 실행: `uv run python -X utf8 ...`.
- `camelCase` 파일/폴더/함수/변수, `PascalCase` 클래스. **파일 삭제 금지** — `_backup/`로 이동.
- `except:` 절대 금지. `except Exception: pass` 금지. 좁힌 예외 + 로깅 + `from exc` 체인.
- 커밋/주석/문서에 AI 흔적 (`AI`, `GPT`, `Claude`, `Generated by`, `Co-Authored-By`) 금지.
- `CLAUDE.md`가 로컬 규칙의 SSOT다. `AGENTS.md`는 `CLAUDE.md`를 먼저 읽으라는 포인터로 유지 — `uv run python -X utf8 docs/skills/ops/tools/syncAgentsMd.py` 사용.
